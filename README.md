Website Description:
"Fruits & Veggies is a simple, user-friendly website developed to showcase fresh fruits and vegetables. It includes a Home page with highlights of fresh products, a Price List page displaying rates and availability, an Order page with a functional order form, and an About page introducing the team and mission. The website is designed with HTML, CSS, and basic JavaScript for interactive features, making it a clean and responsive platform for users to explore and place orders."
